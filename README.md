# sys1.py
![test](https://github.com/Akari0709/robosys2025_1/actions/workflows/test.yml/badge.svg)
- このソフトウェアパッケージは、3条項BSDライセンスの下、再頒布および利用が許可されます。
- このプログラムは、標準入力から受け取った文字列の中に含まれる英字の出現頻度をカウントするツールです。
- 結果は'freq.txt'に書き出され、出現頻度順に上位26件が表示されます。
## 使用方法
- 入力: python3 sysk1.py
- 実行後、次のように表示されます：
  - 文字を入力してください（Ctrl+D で終了）：
- 文字を入力し、入力が終わったら Ctrl + D を押して終了します。
- 実行すると、同じフォルダに'freq.txt'が作成されます。
  - 例:
    -   o 4
    -   e 3
    -   a 2
    -   b 1
    -   c 1
## 必要なソフトウェア
- Python
  - テスト済みバージョン: 3.7 ~ 3.10
